# 🎉 Seasons of Code (SoC) - Official Website

A modern, fully-responsive React.js website for the **Seasons of Code** programming initiative by the **Competitive Coding and Programming Club (CcpC)**.

---

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev

# 3. Open browser to http://localhost:5173
```

That's it! The website is now running. 🎊

## 🔧 Customization

### Change Colors
Edit `tailwind.config.js` - update the colors section

### Update Content
Edit files in `src/pages/` - all text is in the component JSX

### Add Navigation Link
Edit `src/components/Navbar.jsx` - update the navLinks array

### Modify Form
Edit `src/pages/Register.jsx` - add/remove form fields
