## ⚡ 5-Minute Quick Start

### 1. Open Terminal
```bash

```

```bash
npm install
```
*(This takes 2-3 minutes)*

### 3. Start the Website
```bash
npm run dev
```

### 4. Open in Browser
- Browser should open automatically
- If not, visit: **http://localhost:5173**

### 5. Explore!
- Click through all 5 pages
- Check mobile view (resize window)
- Test the forms

**That's it! 🎉 Your website is running!**

## 🎯 Quick Tasks

### View the Website
```bash
npm run dev
```
Opens at http://localhost:5173

### Change Colors
1. Open `tailwind.config.js`
2. Find the `colors` section
3. Update the hex codes
4. Save and refresh browser

### Edit Page Content
1. Open file in `src/pages/` (e.g., `Home.jsx`)
2. Edit text directly
3. Save and see changes instantly

### Add a Navigation Link
1. Open `src/components/Navbar.jsx`
2. Update the `navLinks` array
3. Add your new route

### Build for Production
```bash
npm run build
```
Creates optimized `dist/` folder
